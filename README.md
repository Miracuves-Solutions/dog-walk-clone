## How It Works

1. Pet parent creates a profile for their pet with details and preferences
2. Pet parent searches for walkers and books a walk or sitting session
3. Walker accepts and arrives at the scheduled time
4. GPS tracking records the walk route and duration in real time
5. Session completes; pet parent receives a report with photos
6. Payment is released to the walker minus platform commission
